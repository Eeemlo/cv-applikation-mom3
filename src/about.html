Om uppgiften

Uppgiften bestod av två delar där den första delen gick ut på att skapa en webbtjänst med NodeJs, Express och MongoDB som databas. Webbtjänsten ska kunna hantera CRUD-operationerna CREATE, READ, UPDATE och DELETE samt tillåta Cross Origin Requests.


Den andra delen gick ut på att skapa en webbapplikation som konsumerar den webbtjänst jag skapat i tidigare moment. På webbplatsen används Fetch API för att hämta, skicka samt radera data från och till webbtjänsten.

Slutsatser

Roligt att få testa på att skapa en NoSQL-databas med MongoDB och lärorikt att få repetera användning av CRUD-operationer för att hantera webbtjänsten. Detta moment upplevde jag som relativt simpelt då det gick att återanvända både webbtjänsten och webbapplikationen från föregående moment och anpassa dem till denna uppgift.
